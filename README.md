# 🥚🍅 Детекция перепелиных яиц и помидоров черри

**Автоматическая система подсчёта объектов** для обнаружения перепелиных яиц и помидоров черри на фотографиях со сложным фоном и различным освещением с использованием современных технологий обнаружения объектов.

---

## О проекте

Данный проект реализует интеллектуальный пайплайн анализа изображений, способный обнаруживать, детектировать и подсчитывать перепелиные яйца и помидоры черри (красные и жёлтые) на фотографиях с разнообразными фонами и условиями освещения. Система сочетает **нейросетевое обнаружение** с **классическими методами постобработки**, достигая высокой производительности без необходимости размеченных обучающих данных.

### Ключевые возможности

- **Открытая словарная детекция** на основе Grounding DINO – обнаружение объектов по текстовому описанию, без дообучения
- **Ансамблирование промптов** – множество текстовых описаний (например, "еда", "помидор", "яйцо", "круглый объект") для максимальной полноты
- **Интеллектуальное подавление дубликатов** – удаление пересекающихся детекций с использованием алгоритма пересечения по минимальной площади (IOM)
- **Автоматическое разделение классов** – различение помидоров и яиц
- **Удобный интерфейс** – загрузка изображений через GUI, мгновенный подсчёт с визуальными аннотациями
- **Оптимизация гиперпараметров** – с использованием байесовской оптимизации Optuna для автоматической настройки порогов детекции

---

## Что делает система

| **Входные данные** | **Результат** |
|-----------|------------|
| Любое изображение с перепелиными яйцами и/или помидорами черри | Точный подсчёт яиц и помидоров с визуализацией ограничивающих рамок |
| Сложные фоны (однородные или текстурированные) | Надёжное обнаружение независимо от сложности фона |
| Различные условия освещения (тени, блики) | Стабильная работа при разной освещённости |

## Результаты

| Метрика | Значение |
|--------|-------|
| **Наилучший loss на валидации** | **0.69** (улучшение на 40% по сравнению с ручной настройкой) |
| **Фреймворк детекции** | Grounding DINO |

Система достигает почти полного качества на изображениях выборки, с редкими ошибками при сильном перекрытии объектов.

---

## Технический подход

1. **Предобработка** – стандартная нормализация для подачи в модель
2. **Мультипромптная детекция** – ансамбль текстовых промптов
3. **Подавление дубликатов** – кастомный IOM-алгоритм подавления дубликатов
4. **Подсчёт классов** – общее количество = уникальные детекции; помидоры = детекции с высоким порогом; яйца = общее количество − помидоры
